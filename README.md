#uzaLEAT - LowEndAITrainer

[![License: UOPL](https://img.shields.io/badge/License-UOPL--1.6.2-blue.svg)](LICENSE)

framework built on vulkan specifically for pre-train AI on low end hardware that have built in gutr interpretator for building your own AI architectures from scratch (you can make anything, from basic RNN/SNN/rwkv to something that you invented by yourself).

written on CPP, beta version.
